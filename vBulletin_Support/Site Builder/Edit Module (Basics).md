1. Click "Edit Site" to enable [[Site Builder]]. (if necessary)  
2. Visit the page you want to edit.  
3. Click Edit Page.  
4. Click the pencil icon for the module you want to edit.  
5. Change the options.  
6. Save the [[Modules]].  
7. Save the Page.  
8. Turn of Edit Site if desired.

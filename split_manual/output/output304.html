BB Codes, sometimes referred to as vB Codes, are a used to replace the need for html tags when formatting text in posts.

Examples:

In html to make text bold, italics or underlined you would use:
<b>Bold</b>
<i>Italic</i>
<u>Underlined</u>
With BB Code you use:

[b]Bold[/b]
[i]Italic[/i]
[u]Underlined[/u]
